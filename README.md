📱 Aposentela

Aposentela é uma aplicação web desenvolvida para facilitar a comunicação e interação social de idosos por meio de uma rede social adaptada às suas necessidades, oferecendo uma interface simples, acessível e intuitiva.

A plataforma foi criada com o objetivo de melhorar a usabilidade das redes sociais para o público da terceira idade, reduzindo dificuldades de navegação e tornando a experiência digital mais clara, confortável e acessível, utilizando interfaces simplificadas, botões visíveis, textos maiores e orientações guiadas.

🎯 Objetivo

O principal objetivo do Aposentela é melhorar a acessibilidade e a experiência do usuário idoso em redes sociais, proporcionando uma navegação mais simples, organizada e intuitiva.
