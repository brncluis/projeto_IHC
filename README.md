📱 Aposentela

Aposentela é uma aplicação web desenvolvida para facilitar a comunicação e interação social de idosos por meio de uma rede social adaptada às suas necessidades, oferecendo uma interface simples, acessível e intuitiva.

A plataforma foi criada com o objetivo de melhorar a usabilidade das redes sociais para o público da terceira idade, reduzindo dificuldades de navegação e tornando a experiência digital mais clara, confortável e acessível, utilizando interfaces simplificadas, botões visíveis, textos maiores e orientações guiadas.

🎯 Objetivo

O principal objetivo do Aposentela é melhorar a acessibilidade e a experiência do usuário idoso em redes sociais, proporcionando uma navegação mais simples, organizada e intuitiva.

👥 Público-alvo

O sistema é voltado principalmente para idosos e pessoas com pouca familiaridade com tecnologia, que enfrentam dificuldades ao utilizar redes sociais tradicionais.

⚙️ Tecnologias utilizadas

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript, com apoio do VS Code para desenvolvimento.

🖥️ Funcionalidades

A aplicação conta com funcionalidades básicas de uma rede social, como login de usuário, visualização de publicações, criação de posts, sistema de mensagens e perfil do usuário.

🧪 Testes de usabilidade

Foram realizados testes para avaliar a facilidade de uso da aplicação, considerando o tempo para execução de algumas tarefas e a capacidade dos usuários de concluir as ações propostas.

Os tempos médios observados foram:
Login: 10 segundos
Encontrar botão de publicação: 20 segundos
Encontrar mensagens: 5 segundos
Acessar perfil: 5 segundos

De forma geral, todos os usuários conseguiram concluir as tarefas, indicando que a aplicação é funcional e compreensível. No entanto, foi percebida uma maior dificuldade na localização do botão de publicação, o que indica a necessidade de melhorias nessa parte da interface.

▶️ Execução do projeto

Para executar o projeto, basta baixar ou clonar o repositório e abrir a pasta no VS Code (ou outro editor de sua preferência).

Em seguida, abra o arquivo “index.html” diretamente no navegador. A aplicação funciona de forma local, sem necessidade de instalação de dependências adicionais.
